### 遊戲說明

這是一款射擊遊戲，支援玩家與多台電腦對戰。

* **生命值（HP）：**

  * 玩家：3
  * 電腦：1

* **操作按鍵：**

  * **Z 鍵：** 重新開始遊戲
  * **空白鍵（Space）：** 普通射擊
  * **Ctrl 鍵：** 特殊射擊
