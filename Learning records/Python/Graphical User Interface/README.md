# Graphical User Interface

> ch9 圖形化介面 (Tkinter)
- 建立tkinter視窗
- GUI元件
  - label(文字標籤)
  - Button(按鈕)
  - Entry(文字方塊)
  - Text(文字區塊)
  - Radiobutton(選項按鈕)
  - Checkbutton(核取按鈕)
  - Canvas(畫布)
  - Menu(功能表)
  - messagebox(對話方塊)
  - Listbox(列表)
  - Scrollbar(捲軸)
  - 布局
    - pack(side=XXXX)： xxx可以是TOP、 LEFT、 BOTTOM、 RIGHT
    - grid(row=x, column=y)：以行列布局
    - place(x=10,y=50)：以坐標方式布局
  - 布局管理元件 
    - Frame
    - LabelFrame
    - PanedWindow
- Tkinter事件處理
  - 鍵盤與滑鼠
